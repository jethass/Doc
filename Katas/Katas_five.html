Kata Five — Symfony: WebProfiler - DataCollector

Story

As a developer, I need to see the HTTP request and response in the profiler of Symfony. I also need to view the name of the files if there was. In a popin, I need to have the curl request to replay it any time in command line.

  • Use a DataCollector to collect the Request and the Response.
  • Forge the curl request and populate the data.
  • Follow the following templates (by using bootstrap).

Subject

Symfony

WebProfiler DataCollector

Steps

  1. Create a data collector class which extends Symfony\Component\HttpKernel\DataCollector\DataCollector class.

  2. To enable the data collector, add it as a regular service in one of your configuration file, and tag it with data_collector, with the id http. <tag name="data_collector" template="HttpDataCollectorBundle:Collector:http" id="http"/>

  3. Implement the collect() method.

  4. Populate the data with the Request and Response information using the __toString() method.

  5. Populate the data with the Uploaded files names ()comma separated) if there are some.

  6. Forge the Curl request with request's method, url and headers. Add it to the data.

  7. Create a twig view which extends from the WebProfilerBundle:Profiler:layout.html.twig template.

  8. Create the blocks toolbar, head, menu, panel.

  9. Import the bootstrap library in the head block.

  10. To enable the template, add a template attribute to the data_collector tag in your configuration.

  11. Customize the toolbar, head, menu and panel block in order to show the data collected.

  12. Create the icon.html.twig file to add the icon encoded in base64.

  13. Use a modal to show the curl request.

Solution

HttpDataCollector

PHP
<?php

namespace Collector;

use Symfony\Component\HttpKernel\DataCollector\DataCollector;
use Symfony\Component\HttpFoundation\Request;
use Symfony\Component\HttpFoundation\Response;

class HttpDataCollector extends DataCollector
{
    public function collect(Request $request, Response $response, \Exception $exception = null)
    {
        $this->data = array(
            'curl_request' => $this->forgeRequest($request),
            'request' => $request->__toString(),
            'files' => implode(', ', array_map(function($file) { return $file->getClientOriginalName(); } , $request->files->all())),
            'response' => $response->__toString()
        );
    }

    public function getName()
    {
        return 'http';
    }

    public function getCurlRequest()
    {
        return $this->data['curl_request'];
    }

    public function getFiles()
    {
        return $this->data['files'];
    }

    public function getRequest()
    {
        return $this->data['request'];
    }

    public function getResponse()
    {
        return $this->data['response'];
    }

    private function forgeRequest(Request $request)
    {
        $curlRequest = 'curl -X ' . $request->getMethod() . ' http://' . $request->server->get('HTTP_HOST') . $request->getRequestUri();

        foreach ($request->headers as $name => $header) {
            $curlRequest .=  ' -H "'. $name . ': ' . $header[0] .'"';
        }

        return $curlRequest;
    }
}
